# 🛒 Shopsphere

Shopsphere is a responsive, feature-rich e-commerce front-end designed to provide a seamless shopping experience. It includes product browsing, a shopping cart interface, and an integrated order tracking system to keep customers updated on their purchases.

## ✨ Features

* **Responsive Design:** Fully optimized for desktop, tablet, and mobile viewing.
* **Product Showcase:** Clean and modern product grids with hover effects and detailed views.
* **Order Tracking System:** Visual stepper interface allowing users to see real-time updates on their package (Placed -> Shipped -> Out for Delivery -> Delivered).
* **Intuitive UI/UX:** Built with modern web design principles to ensure easy navigation and a smooth checkout flow.

## 🛠️ Tech Stack

* **HTML5:** Semantic markup for structure and accessibility.
* **CSS3:** Custom styling, flexbox/grid layouts, and animations (no heavy frameworks required).
* **JavaScript (ES6+):** Dynamic interactions, cart logic, and order tracking UI updates.
